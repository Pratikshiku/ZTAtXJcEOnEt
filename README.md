### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户(客户)、销售经理、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 房产信息
5. 论坛
6. 公告信息
7. 个人中心
8. 房产收藏


**销售经理功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 房产信息管理、房产信息收藏管理
6. 预约看房管理
7. 论坛管理
8. 购置房产管理
9. 客户打分管理
10. 公告信息管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 基础数据管理(房产户型管理、公告类型管理)
5. 房产信息管理、房产信息收藏管理
6. 预约看房管理
7. 论坛管理
8. 购置房产管理
9. 客户打分管理
10. 公告信息管理
11. 客户管理
12. 销售经理管理
13. 轮播图信息
